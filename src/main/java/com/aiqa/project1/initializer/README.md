# initializer

本目录存放应用启动后的初始化逻辑。

## 主要文件

- `SystemInitRunner`: 实现 `CommandLineRunner`，在 Spring Boot 启动后执行系统初始化任务。

## 当前职责

`SystemInitRunner` 会注入用户、标签、向量库、密码编码器、环境配置和缓存工具等依赖，用于在应用启动阶段准备默认数据或同步系统状态。

## 维护提示

- 初始化逻辑会影响应用启动成功率，外部服务调用要考虑失败策略。
- 涉及默认管理员、默认标签、Milvus 集合或缓存预热时，应保持幂等。
- 不建议在这里放长时间阻塞任务，耗时任务应交给后台线程或消息队列。
